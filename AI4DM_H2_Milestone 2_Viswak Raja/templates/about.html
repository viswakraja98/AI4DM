Viswak's Project Space



In this space, you'll find a series of progressive classroom assignments focused on creating interactive interfaces that leverage various AI models. The vision of my project is to make creative expression and collaboration more accessible to BLV users. My goal is to provide agency and opportunities for co-creation across different abilities, by creating a digital canvas for creating and modifying artwork. So far, I have explored models from Hugging Face, OpenAI, and LangChain, applying each to develop dynamic and responsive user experiences.



Milestone 1: Initial Development and AI Integration

Begin your journey into text-to-image generation and spatially aware image descriptions with Milestone 1. This phase is dedicated to establishing the foundational technology stack and integrating advanced AI models to generate rich, descriptive, spatially, and contextually aware image outputs.

  • Text to Image: Utilize Hugging Faceâ€™s model to generate images from user prompt.
  • Image Description: Employ OpenAI's GPT-4o mini to craft descriptive alternative text based on the generated image.
  • Interactive Display: The image descriptions are dynamically presented on a P5.js canvas with accompanying narration via the P5.js speech library.
  • Framework: Built using Flask and styled with Bootstrap for a robust and responsive user experience.

Milestone 2: Advanced Features and Enhanced Interactivity

Milestone 2 enhances image descriptions by incorporating the LangChain library to infuse emotional depth into the output. This phase enables the selection of various emotions, allowing users to generate contextually rich alt text that resonates with the chosen emotional tone, making image descriptions more immersive and meaningful.

  • Emotion Selection: Users can choose from six emotions, tailoring the image description to their preferences.
  • Prompt Templates and RAG: Enhance the quality of image descriptions by using structured prompts and contextually enriched content through Retrieval-Augmented Generation (RAG).
  • Customized window: Different background colors match the selected emotions.
